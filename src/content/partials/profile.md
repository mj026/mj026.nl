Ik ben een ervaren software ontwikkelaar met name met de programmeertaal Python. Ik stuur graag aan op leesbare programmacode die goed onderhoudbaar is voor de langere termijn. Ik automatiseer en voorzie QA, CI/CD, DevOps en deployment met diverse tools als Docker, GitLab CI/CD, GitHub Actions, Ansible/Saltstack of Terraform.

<br>De laatste jaren heb ik vooral met Django gewerkt, in combinatie met Django REST Framework en/of Wagtail. Front-end ontwikkeling met Javascript/Typescript, (S)CSS of frameworks zoals Tailwind.css en Vue.js zijn mij niet vreemd en doe ik ook wanneer het project er om vraagt.

<br>Privacywaardes, security en milieu-impact heb ik hoog in het vaandel staan. De huidige AI trend volg ik op de voet met een realistische en kritische blik.

<br>Ik kan prima zelfstandig aan een project of opdracht werken. In projectteams kan ik allerlei rollen vervullen zoals (lead-)developer, systeemontwerper, consultant, reviewer van code of criticus van technische beslissingen.
